## ERC-20 Edge Cases
<ol>
  <li>Fee-on-transfer</li>
  <li>Callbacks</li>
  <li>Return false VS reverting on failure</li>
  <li>Revert on zero address or 0 amount</li>
  <li>No return value</li>
  <li>Blacklisted addresses</li>
  <li>Subsequent non-zero approvals not allowed</li>
  <li>Multiply addresses for a token</li>
  <li>Upgradeable tokens</li>
  <li>Flash mintable tokens</li>
  <li>Transfer less than amount</li>
  <li>Tokens with lots of decimals</li>
  <li>Pausable tokens</li>
  <li>Revert in large approvals or transfers</li>
</ol>
